# 로또 1단계 - 콘솔 기반 로또 게임

- 본 프로젝트는 페어프로그래밍으로 진행된다.
- 기능을 추가할 떄 마다 기능 요구 사항을 업데이트 한다.

## 기능 요구 사항

### 구입 금액

- [x] 로또 구입 금액을 입력받는다.
- [x] 구입한 로또 개수를 출력한다.

### 로또 번호 생성

- [x] 구입한 개수만큼 로또를 발행한다.

### 당첨 번호 및 보너스 번호 비교

- [x] 구매한 로또 번호와 당첨 번호를 비교한다.
- [x] 구매한 로또 번호와 보너스 번호를 비교한다.

### 당첨 내역

- [x] 당첨 내역을 계산한다.
- [x] 당첨 내역을 출력한다.

### 총 수익률

- [x] 총 수익률을 계산한다.
- [x] 총 수익률을 출력한다.

### 다시 시작하기

- [x] y를 입력하면 로또 게임을 다시 시작한다.
- [x] n을 입력하면 로또 게임을 종료한다.

# 로또 2단계 - 웹 기반 로또 게임

### 마크업 및 UI

- [x] 디자인 시스템 정의
- [x] 아이콘 에셋 추가
- [x] HTML 마크업
- [x] 공통 컴포넌트 UI 구현 (button, input)
- [x] 레이아웃 및 세부 UI 구현
- [x] 모달 구현

### 구입 금액 입력 및 로또 발행

- [x] 구입할 금액을 입력하고 '확인' 버튼을 누르면 로또를 발행한다.
- [x] 발행된 로또를 화면에 렌더링한다.
- [x] 로또가 이미 발행된 상태에서 다시 구입한다면 기존 발행된 로또를 초기화하고 다시 발행한다.

### 보너스 / 당첨 번호 입력 및 통계 출력

- [x] 구매한 로또 번호와 당첨 번호, 보너스 번호를 비교한다.
- [x] '결과 확인하기' 버튼을 누르면 당첨 통계 모달을 렌더링한다.
  - [x] 당첨 내역을 출력한다.
  - [x] 수익률을 출력한다.
- [x] 당첨 통계가 이미 출력된 상태에서 다시 '결과 확인하기' 버튼을 누른면 기존 통계를 초기화하고 다시 출력한다.

### 다시 시작하기

- [x] 당첨 통계 모달에서 '다시 시작하기 '버튼을 누르면 모달을 닫는다.
- [x] 모든 상태를 초기화한다.
